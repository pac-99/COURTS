About CourtCheck

CourtCheck helps players instantly see how busy local sports courts are by combining crowdsourced reports and AI-enhanced camera data.

Mission

To eliminate the frustration of showing up to crowded courts by providing real-time availability information through community-driven data and smart technology.

How It Works

  1. Community Reports: Users check in and report crowd levels with photos
  2. AI Analysis: Cameras automatically count players and estimate occupancy
  3. Smart Integration: Combine user data with AI data for accurate status
  4. Real-time Updates: Status updates instantly across the platform

Tech Stack

  • Backend: Node.js, Express, MongoDB
  • Frontend: Vanilla JavaScript, Google Maps API
  • AI/ML: TensorFlow Lite, OpenCV
  • Maps: Google Maps with geospatial indexing
